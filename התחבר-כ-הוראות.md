# התחברות כאדמין כמשתמש אחר (לפי UID)

כפתור קטן "Login as..." בתחתית מסך ההתחברות – לאדמין בלבד.

## שימוש

1. פתחי את מסך ההתחברות (לפני התחברות)
2. גללי למטה – תראי "Login as..."
3. לחצי, הדביקי את ה-UID (מ-Firebase Console → Authentication → Users)
4. לחצי "התחבר"

---

## הגדרה חד־פעמית

### 1. סוד ב-Firebase

```bash
firebase functions:config:set admin.login_secret="הסוד-שלך"
firebase deploy --only functions
```

### 2. פריסת כללי Firestore (חשוב – בלי זה תהיה "permission-denied")

```bash
firebase deploy --only firestore:rules
```

### 3. בנייה עם הסוד (הסוד לא נשמר ב-Git)

```bash
flutter build web --dart-define=ADMIN_LOGIN_SECRET=הסוד-שלך
firebase deploy --only hosting
```

להריצה מקומית:
```bash
flutter run -d chrome --dart-define=ADMIN_LOGIN_SECRET=הסוד-שלך
```
